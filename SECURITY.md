# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.3   | :white_check_mark: |
| < 0.2.3 | :x:                |

## Reporting a Vulnerability

Please [report vulnerabilities](https://github.com/coignard/o2/security/advisories/new) rather than opening a public issue. Thanks!
